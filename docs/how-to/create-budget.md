# How to Create a Budget

This guide walks you through creating a budget using forecast data and assumptions.

---

## Before you begin

Ensure that:
- You have access to the planning workspace
- Required dimensions (time, department, account) are available
- A baseline forecast or actuals exist

---

## Step 1: Create a new budget version

1. Navigate to **Planning > Budgets**
2. Select **Create New Version**
3. Enter a name and description
4. Choose the time period for the budget

---

## Step 2: Select a starting point

You can initialize a budget using:
- Prior year actuals
- Latest forecast version
- A blank template

Select the option that best aligns with your planning approach.

---

## Step 3: Apply assumptions

Apply budget assumptions such as:
- Percentage growth or reduction
- Headcount changes
- Seasonal adjustments

Assumptions can be applied globally or at a granular level.

---

## Step 4: Review AI insights (optional)

If enabled, AI insights may:
- Flag values that deviate significantly from historical trends
- Highlight areas requiring closer review

Use these insights to validate assumptions.

---

## Step 5: Submit for approval

Once complete:
1. Review totals and summaries
2. Submit the budget for approval
3. Track approval status within the workflow

---

## Next steps

After approval, budgets can be:
- Locked for reporting
- Compared against actuals
- Used as inputs for future forecasts
