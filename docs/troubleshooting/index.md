# Troubleshooting

This section helps you resolve common issues related to forecasting and budgeting.

---

## Forecast values are not updating

**Possible causes:**
- The forecast version is locked
- Required drivers are missing
- You do not have edit permissions

**Resolution:**
- Verify version status
- Check driver configuration
- Confirm your access level

---

## AI insights are not visible

**Possible causes:**
- AI features are disabled for your role
- Insufficient historical data
- Data refresh is incomplete

**Resolution:**
- Contact an administrator
- Ensure sufficient historical data exists
- Retry after data refresh

---

## Approval workflow is blocked

**Possible causes:**
- Pending approvals
- Missing required reviewers

**Resolution:**
- Review workflow status
- Ensure all required approvers are assigned

---

## Performance issues when opening large forecasts

**Possible causes:**
- Large dimensionality
- Concurrent user activity

**Resolution:**
- Filter data where possible
- Work with administrators to optimize models
